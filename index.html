Insira a quantidade para cada item de saber e competência para calcular sua pontuação total de RSC.

Resumo da Pontuação

Pontuação Total Atual: Este é o somatório de todos os pontos obtidos pelos saberes e competências que você preencheu. Cada item tem uma pontuação base, que é multiplicada pela quantidade informada.

0

Itens Contabilizados: Indica a quantidade de saberes e competências diferentes para os quais você informou uma quantidade maior que zero. Para atingir os Níveis RSC, é necessário um mínimo de itens contabilizados.

0

Requisitos Mínimos por Nível de RSC Os Níveis RSC (Reconhecimento de Saberes e Competências) são a forma de progredir na carreira, equivalente a ter títulos acadêmicos. Para cada nível, você precisa de uma Pontuação Mínima e um Mínimo de Itens Contabilizados.

A caixa estará verde se você atendeu aos requisitos de pontuação e itens para aquele nível.

A caixa estará vermelha se você ainda não atendeu a um ou ambos os requisitos.
